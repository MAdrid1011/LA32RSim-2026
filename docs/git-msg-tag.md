# Git Commit Message Tags

- `[sim]`: Changes to the Verilator simulation host (sim/ directory).
- `[ref]`: Changes to the reference simulator (ref/ directory).
- `[rtl]`: Changes to the RTL design files (rtl/ directory).
- `[software]`: Changes to test software (software/ directory).
- `[build]`: Changes to build scripts, Makefiles, or Kconfig.
- `[docs]`: Changes to documentation only.
- `[fix]`: Bug fixes that do not fall under another specific tag.
- `[feat]`: New features spanning multiple components.
